# Planning Docs

This directory is for planning documents that are useful while TARR work is
being designed or implemented.

## Layout

- `archive/`: completed or superseded plans kept for historical context.

## Policy

- Keep active, actionable plans at this level only while they are being used.
- Move finished, superseded, or implementation-specific plans to `archive/`.
- Prefer updating canonical docs under `docs_my/TARR/` once a plan becomes the
  current implementation contract.
- Keep archived plans immutable except for link fixes, title fixes, and archive
  index maintenance.
